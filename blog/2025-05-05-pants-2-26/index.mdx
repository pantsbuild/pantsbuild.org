We are pleased to announce Pants [2.26.0](https://github.com/pantsbuild/pants/blob/2.26.x/docs/notes/2.26.x.md), the latest release of [Pantsbuild, the scalable and ergonomic build system](https://www.pantsbuild.org/). To update, set `pants_version = "2.26.0"` in your `pants.toml`. If you're not using Pants yet, [get started now](https://www.pantsbuild.org/2.26/docs/getting-started).

Thank you to [Klaviyo](https://www.klaviyo.com/) for their Platinum tier support throughout this release!
