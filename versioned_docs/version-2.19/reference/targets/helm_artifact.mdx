## `tags`

<Field
    type_repr={`Iterable[str] | None`}
    default_repr={`None`}
>

Arbitrary strings to describe a target.

For example, you may tag some test targets with &#x27;integration_test&#x27; so that you could run `scie-pants-linux-x86_64 --tag='integration_test' test ::` to only run on targets with that tag.

</Field>

## `description`

<Field
    type_repr={`str | None`}
    default_repr={`None`}
>

A human-readable description of the target.

Use `scie-pants-linux-x86_64 list --documented ::` to see all targets with descriptions.

</Field>

## `registry`

<Field
    type_repr={`str | None`}
    default_repr={`None`}
>

Either registry alias (prefixed by `@`) configured in `[helm.registries]` for the Helm artifact or the full OCI registry URL.

</Field>

<div aria-hidden="true" style={ { visibility: 'hidden', height: 0 } }>

## `repository`

<Field
    type_repr={`str | None`}
    default_repr={`None`}
>

Either a HTTP(S) URL to a classic repository, or a path inside an OCI registry (when `registry` is provided).

</Field>
