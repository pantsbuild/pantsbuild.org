```
pants dependees [args]
```

List all targets that depend on any of the input files/targets.

Backend: <span className="color--primary">`pants.core`</span>

Config section: <span className="color--primary">`[dependees]`</span>

<span className={styles.options}>

## Basic options

### `output_file`

<Option
  cli_repr={`--dependees-output-file=<path>`}
  env_repr='PANTS_DEPENDEES_OUTPUT_FILE'
  default_repr={`None`}
>

Output the goal&#x27;s stdout to this file. If unspecified, outputs to stdout.

</Option>

### `sep`

<Option
  cli_repr={`--dependees-sep=<separator>`}
  env_repr='PANTS_DEPENDEES_SEP'
  default_repr={`\\n`}
>

String to use to separate lines in line-oriented output.

</Option>

### `transitive`

<Option
  cli_repr={`--[no-]dependees-transitive`}
  env_repr='PANTS_DEPENDEES_TRANSITIVE'
  default_repr={`False`}
>

List all transitive dependees. If unspecified, list direct dependees only.

</Option>

### `closed`

<Option
  cli_repr={`--[no-]dependees-closed`}
  env_repr='PANTS_DEPENDEES_CLOSED'
  default_repr={`False`}
>

Include the input targets in the output, along with the dependees.

</Option>
