A Rust package as defined in https://doc.rust-lang.org/book/ch07-01-packages-and-crates.html.

Expects that there is a `Cargo.toml` target in its root directory

Backend: <span className="color--primary">`pants.backend.experimental.rust`</span>

---

<span className={styles.options}>

## `description`

<Field
    type_repr={`str | None`}
    default_repr={`None`}
>

A human-readable description of the target.

Use `pants list --documented ::` to see all targets with descriptions.

</Field>

## `skip_rustfmt`

<Field
    type_repr={`bool`}
    default_repr={`False`}
    backend="pants.backend.experimental.rust"
>

If true, don&#x27;t run rustfmt on this crate.

</Field>

## `sources`

<Field
    type_repr={`Iterable[str] | None`}
    default_repr={`('src/**/*.rs', 'tests/**/*.rs')`}
>

A list of files and globs that belong to this target.

Paths are relative to the BUILD file&#x27;s directory. You can ignore files/globs by prefixing them with `!`.

Example: `sources=['src/main.rs', 'src/new_*.rs', '!src/old_ignore.py']`

</Field>

## `tags`

<Field
    type_repr={`Iterable[str] | None`}
    default_repr={`None`}
>

Arbitrary strings to describe a target.

For example, you may tag some test targets with &#x27;integration_test&#x27; so that you could run `pants --tag='integration_test' test ::` to only run on targets with that tag.

</Field>

</span>
