Options used to configure and launch a Debug Adapter server.

See https://microsoft.github.io/debug-adapter-protocol/ for more information.

Backend: <span className="color--primary">`pants.core`</span>

Config section: <span className="color--primary">`[debug-adapter]`</span>

<span className={styles.options}>

## Basic options

### `host`

<Option
  cli_repr={`--debug-adapter-host=<str>`}
  env_repr='PANTS_DEBUG_ADAPTER_HOST'
  default_repr={`127.0.0.1`}
>

The hostname to use when launching the server.

</Option>

<Option
  cli_repr={`--debug-adapter-port=<int>`}
  env_repr='PANTS_DEBUG_ADAPTER_PORT'

The port to use when launching the server.

</Option>
