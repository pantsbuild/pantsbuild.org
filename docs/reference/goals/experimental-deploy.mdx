Perform a deployment process.

Backend: <span className="color--primary">`pants.core`</span>

Config section: <span className="color--primary">`[experimental-deploy]`</span>

<span className={styles.options}>

## Basic options

### `dry_run`

<Option
  cli_repr={`--[no-]experimental-deploy-dry-run`}
  env_repr='PANTS_EXPERIMENTAL_DEPLOY_DRY_RUN'
  toml_repr={`[experimental-deploy]
dry_run = <bool>`}
  default_repr={`False`}
>

If true, perform a dry run without deploying anything. For example, when deploying a terraform_deployment, a plan will be executed instead of an apply.

</Option>

### `publish_dependencies`

<Option
  cli_repr={`--[no-]experimental-deploy-publish-dependencies`}
  env_repr='PANTS_EXPERIMENTAL_DEPLOY_PUBLISH_DEPENDENCIES'
  toml_repr={`[experimental-deploy]
publish_dependencies = <bool>`}
  default_repr={`True`}
>

If false, don&#x27;t publish target dependencies before deploying the target. For example, when deploying a helm_deployment, dependent docker images will not be published.

</Option>
