A plugin for `scalac`.

Currently only thirdparty plugins are supported. To enable a plugin, define this target type, and set the `artifact=` field to the address of a `jvm_artifact` that provides the plugin.

If the `scalac`-loaded name of the plugin does not match the target&#x27;s name, additionally set the `plugin_name=` field.

Backend: <span className="color--primary">`pants.backend.experimental.scala`</span>

<Field
