```
pants repl [args]
```

Open a REPL with the specified code loadable.

Backend: <span className="color--primary">`pants.core`</span>

Config section: <span className="color--primary">`[repl]`</span>

<span className={styles.options}>

## Basic options

### `shell`

<Option
  cli_repr={`--repl-shell=<str>`}
  env_repr='PANTS_REPL_SHELL'
  default_repr={`None`}
>

Override the automatically-detected REPL program for the target(s) specified.

</Option>

### `args`

<Option
  cli_repr={`--repl-args="[<shell_str>, <shell_str>, ...]", ... -- [<shell_str> [<shell_str> [...]]]`}
  env_repr='PANTS_REPL_ARGS'
  default_repr={`[]`}
>

Arguments to pass directly to the repl program, e.g. `--repl-args='-i helloworld/main.py'`.

Currently supported only for the ipython shell.

</Option>

### `restartable`

<Option
  cli_repr={`--[no-]repl-restartable`}
  env_repr='PANTS_REPL_RESTARTABLE'
  default_repr={`False`}
>

True if the REPL should be restarted if its inputs have changed.

</Option>
