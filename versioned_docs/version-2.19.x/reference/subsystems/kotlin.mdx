## Basic options

### `version_for_resolve`

<Option
  cli_repr={`--kotlin-version-for-resolve="{'key1': val1, 'key2': val2, ...}"`}
  env_repr='PANTS_KOTLIN_VERSION_FOR_RESOLVE'
  default_repr={`{}`}
>

A dictionary mapping the name of a resolve to the Kotlin version to use for all Kotlin targets consuming that resolve.

</Option>

## Advanced options

### `tailor_source_targets`

<Option
  cli_repr={`--[no-]kotlin-tailor-source-targets`}
  env_repr='PANTS_KOTLIN_TAILOR_SOURCE_TARGETS'
  default_repr={`True`}
>

If true, add `kotlin_sources` targets with the `tailor` goal.

</Option>
