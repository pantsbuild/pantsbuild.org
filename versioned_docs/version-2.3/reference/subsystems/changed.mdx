Tell Pants to detect what files and targets have changed from Git.

See https://www.pantsbuild.org/v2.3/docs/advanced-target-selection.

Backend: <span className="color--primary">`pants.core`</span>

Config section: <span className="color--primary">`[changed]`</span>

<span className={styles.options}>

## Basic options

### `since`

<Option
  cli_repr={`--changed-since=<str>`}
  env_repr='PANTS_CHANGED_SINCE'
  default_repr={`None`}
>

Calculate changes since this Git spec (commit range/SHA/ref).

</Option>

### `diffspec`

<Option
  cli_repr={`--changed-diffspec=<str>`}
  env_repr='PANTS_CHANGED_DIFFSPEC'
  default_repr={`None`}
>

Calculate changes contained within a given Git spec (commit range/SHA/ref).

</Option>

### `dependees`

<Option
  cli_repr={`--changed-dependees=<DependeesOption>`}
  env_repr='PANTS_CHANGED_DEPENDEES'
  one_of='none, direct, transitive'
  default_repr={`none`}
>

Include direct or transitive dependees of changed targets.

</Option>
