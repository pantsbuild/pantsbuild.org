## Basic options

### `skip`

<Option
  cli_repr={`--[no-]gofmt-skip`}
  env_repr='PANTS_GOFMT_SKIP'
  default_repr={`False`}
>

If true, don&#x27;t use gofmt when running `scie-pants-linux-x86_64 fmt` and `scie-pants-linux-x86_64 lint`.

</Option>

### `args`

<Option
  cli_repr={`--gofmt-args="[<shell_str>, <shell_str>, ...]"`}
  env_repr='PANTS_GOFMT_ARGS'
  default_repr={`[]`}
>

Arguments to pass directly to gofmt, e.g. `--gofmt-args='-s -e'`.

Only the following style related options are supported: `-e`, `-r`, `-s`.

</Option>
