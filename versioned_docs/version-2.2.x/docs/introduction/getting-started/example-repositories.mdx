Examples to help set up your own repository.

---

- A [Python repo](https://github.com/pantsbuild/example-python), demonstrating features such as:

  - running tests
  - using linters and formatters
  - using MyPy
  - running a REPL
  - building and running PEX files
  - building an AWS Lambda
  - generating `setup.py` and building `.whl` files
  - generating Protocol Buffer code

- A [Pants plugin repo](https://github.com/pantsbuild/example-plugin), demonstrating how to customize Pants with your own functionality by adding support for Bash.
