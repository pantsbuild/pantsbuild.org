<span className={styles.options}>

## Basic options

### `output_file`

<Option
  cli_repr={`--peek-output-file=<path>`}
  env_repr='PANTS_PEEK_OUTPUT_FILE'
  default_repr={`None`}
>

Output the goal&#x27;s stdout to this file. If unspecified, outputs to stdout.

</Option>

### `exclude_defaults`

<Option
  cli_repr={`--[no-]peek-exclude-defaults`}
  env_repr='PANTS_PEEK_EXCLUDE_DEFAULTS'
  default_repr={`False`}
>

Whether to leave off values that match the target-defined default values.

</Option>

### `include_dep_rules`

<Option
  cli_repr={`--[no-]peek-include-dep-rules`}
  env_repr='PANTS_PEEK_INCLUDE_DEP_RULES'
  default_repr={`False`}
>

Whether to include `_dependencies_rules`, `_dependents_rules` and `_applicable_dep_rules` that apply to the target and its dependencies.

</Option>
