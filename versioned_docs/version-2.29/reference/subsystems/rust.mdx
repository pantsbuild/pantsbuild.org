## Basic options

### `rustup_search_paths`

<Option
  cli_repr={`--rust-rustup-search-paths="['<str>', '<str>', ...]"`}
  env_repr='PANTS_RUST_RUSTUP_SEARCH_PATHS'
  toml_repr={`[rust]
rustup_search_paths = [
    '<str>',
    '<str>',
    ...,
]`}
  default_repr={`[\n  "<PATH>"\n]`}
>

A list of paths to search for Rustup.

Specify absolute paths to directories with the `rustup` binary, e.g. `/usr/bin`. Earlier entries will be searched first. The special string &#x27;&#x3C;PATH&#x3E;&#x27; will expand to the contents of the PATH env var.

</Option>

### `toolchain`

<Option
  cli_repr={`--rust-toolchain=<str>`}
  env_repr='PANTS_RUST_TOOLCHAIN'
  toml_repr={`[rust]
toolchain = <str>`}
  default_repr={`stable`}
>

Name of a Rust toolchain to use for all builds. The toolchain name will be provided to Rustup to find the Toolchain.

</Option>
