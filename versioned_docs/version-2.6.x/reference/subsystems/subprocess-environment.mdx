## Advanced options

### `env_vars`

<Option
  cli_repr={`--subprocess-environment-env-vars="['<str>', '<str>', ...]"`}
  env_repr='PANTS_SUBPROCESS_ENVIRONMENT_ENV_VARS'
  default_repr={`[\n  "LANG",\n  "LC_CTYPE",\n  "LC_ALL"\n]`}
>

Environment variables to set for process invocations. Entries are either strings in the form `ENV_VAR=value` to set an explicit value; or just `ENV_VAR` to copy the value from Pants&#x27;s own environment.

</Option>
