## Deprecated options

### `providers`

  cli_repr={`--basicauth-providers="{'key1': val1, 'key2': val2, ...}"`}

  default_repr={`{}`}
  removal_version='2.1.0.dev0'
  removal_hint={'The option `--basicauth-provides` does not do anything and the `[basicauth]` subsystem will be removed.'}

Map from provider name to config dict. This dict contains the following items: &#123;provider_name: &lt;url of endpoint that accepts basic auth and sets a session cookie&gt;&#125;. For example, `{'prod': 'https://app.pantsbuild.org/auth'}`.

</Option>
