Tell Pants to detect what files and targets have changed from Git.

See https://www.pantsbuild.org/v2.16/docs/advanced-target-selection.

Backend: <span className="color--primary">``</span>

Config section: <span className="color--primary">`[changed]`</span>

<span className={styles.options}>

## Basic options

### `since`

<Option
  cli_repr={`--changed-since=<str>`}
  env_repr='PANTS_CHANGED_SINCE'
  toml_repr={`[changed]
since = <str>`}
  default_repr={`None`}
>

Calculate changes since this Git spec (commit range/SHA/ref).

</Option>

### `diffspec`

<Option
  cli_repr={`--changed-diffspec=<str>`}
  env_repr='PANTS_CHANGED_DIFFSPEC'
  toml_repr={`[changed]
diffspec = <str>`}
  default_repr={`None`}
>

Calculate changes contained within a given Git spec (commit range/SHA/ref).

</Option>

### `dependents`

<Option
  cli_repr={`--changed-dependents=<DependentsOption>`}
  env_repr='PANTS_CHANGED_DEPENDENTS'
  toml_repr={`[changed]
dependents = <DependentsOption>`}
  one_of='none, direct, transitive'
  default_repr={`none`}
>

Include direct or transitive dependents of changed targets.

</Option>
