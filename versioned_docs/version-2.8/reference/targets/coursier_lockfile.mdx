## `source`

<Field
    type_repr={`str | None`}
    default_repr={`None`}
>

A single Pants Coursier Lockfile source.

Use `./pants coursier-resolve ...` to generate (or regenerate) the Lockfile. If the Lockfile doesn&#x27;t exist on disk, the first run of `coursier-resolve` will attempt to generate it for you to the default file name (&#x27;coursier_resolve.lockfile&#x27;). After running `coursier-resolve` for the first time, you should update this field&#x27;s`sources` to explicit take ownership of the generated lockfile.
